## Quick Start

```bash
git clone https://github.com/josezuma/binary-converter.git
cd binary-converter
python3 scripts/binary_converter.py --help
```

## Demo

```bash
$ python3 scripts/binary_converter.py

binary-converter v1.0.0
```
